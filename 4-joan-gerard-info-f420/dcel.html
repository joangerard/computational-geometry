HW4: DCEL Data Structure Implementation

  • Click on an edge will add a vertex.
  • To select a vertex just click on it. Red means it is selected. Black means it is not selected. To unselect a vertex click on it again (it should turn to black again).
  • Click on two vertices will add an edge between both vertices.
  • Notice that the operation of adding an edge directly by clicking on a vertex and then on an edge (and viceversa) is not supported.
  • Source code is found in this repo.
